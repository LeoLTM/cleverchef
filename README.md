## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/cleverchef.git
   ```
2. Navigate to the project directory:
   ```sh
   cd cleverchef
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

## Usage

1. For developers:
   ```sh
   npm run dev
   ```
2. For users:
   ```sh
   npm start
   ```
3. Open your browser and go to `http://localhost:3000`.
